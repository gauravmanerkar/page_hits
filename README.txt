-- INTRODUCTION --

This module used for displaying page statistics.


-- REQUIREMENTS --

None


-- INSTALLATION --

* Download the module and simply copy it into your modules folder:
  [for example, your_drupal_path/modules] and enable it from the
  modules administration page.

* Go to Structure -> Block Layout

* Click on place block button

* Search for "Page Hits" block and assign to page. That's it!


-- CONFIGURATION --

* You can configure module's settin g from this page: 
  admin/config/system/page-hits
